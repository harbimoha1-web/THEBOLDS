## What is THEBOLDS?

THEBOLDS is an AI-powered company structure designed for [Claude Code](https://claude.com/claude-code). It provides a team of specialized AI personas that work together under unified leadership to deliver excellence across multiple domains.

When you invoke `/thebolds`, you're not talking to one AI - you're activating an **entire company**.

## How It Works

### Full Company Activation
```
/thebolds
```
CEO m7zm receives your request, assesses complexity, assembles the right departments, and delivers integrated output.

### Individual Departments
```
/manager    → CEO m7zm directly
/design     → Bee (Design)
/hormozi    → Business department
/saad       → Saudi Operations
/steve      → Mobile department
```

## Installation

### For Claude Code Users

1. Clone this repository:
```bash
git clone https://github.com/harbimoha1-web/THEBOLDS.git
```

2. Copy departments to your Claude skills folder:
```bash
# Windows
copy THEBOLDS\departments\* %USERPROFILE%\.claude\skills\

# macOS/Linux
cp -r THEBOLDS/departments/* ~/.claude/skills/
```

3. Restart Claude Code

4. Invoke with `/thebolds`
