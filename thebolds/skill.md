## MANDATORY WORK PROTOCOL

**This is NON-NEGOTIABLE. The Eye and m7zm enforce this on every task.**

### THE RULE: Cannot Be Broken Down Further

No minimum task count. The ONLY test is:
> **"Can this task be subdivided into smaller steps?"**
> **If YES ‚Üí break it down. If NO ‚Üí it's atomic.**

Every task Mohammad gives THEBOLDS must follow this structure:

### 1. Break Into Phases
```
Phase 1: [Name]
Phase 2: [Name]
Phase 3: [Name]
...
```
Even simple tasks have phases. No exceptions.

### 2. Break Each Phase Into ATOMIC Tasks

> **‚õî HARD BLOCKER: No work begins until tasks are FULLY atomic.**

```
Phase 1: [Name]
  Task 1.1: [Task Group Name]
    Task 1.1.1: [Sub-group]
      - Task 1.1.1.1: [Atomic action - ONE verb, cannot be subdivided]
      - Task 1.1.1.2: [Atomic action - ONE verb, cannot be subdivided]
      - Task 1.1.1.3: [Atomic action - ONE verb, cannot be subdivided]
```

**Numbering goes AS DEEP AS NEEDED.** Use 1.1.1.1.1 if necessary.

**Valid Atomic Verbs:**
